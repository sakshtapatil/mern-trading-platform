# MERN Trading Platform 📈

A stock trading platform inspired by Zerodha, built with MongoDB, Express, React and Node.js.

## 🌐 Live Demo
- **Landing Page:** [mern-tradingplatform.netlify.app](https://mern-tradingplatform.netlify.app)
- **Dashboard:** [merntrading.netlify.app](https://merntrading.netlify.app)

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | React.js, Axios, React Toastify |
| Dashboard | React.js, Material UI, Chart.js |
| Backend | Node.js, Express.js, MongoDB, JWT |
| Deployment | Netlify, Render, MongoDB Atlas |
